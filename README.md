

## How to use

1. Clone this repository.
```
git clone https://github.com/Kakhramon/resume.git
```

2. Go to the cloned directory (e.g. `cd Resume`).

3. Run `npm install`.

4. Customize your resume

5. Preview resume with `npm run serve`.

6. Build it with `npm run build`.
